# Projekteksamen - HA(it) - CBS Forår 2026.

### .env konfiguration

Opret en fil med navnet `.env` i root mappen (ikke src/) med vores secrets fra Appendix 9.4

```env
DB_SERVER=
DB_NAME=
DB_USER=
DB_PASSWORD=
DB_PORT=

DATAFORDELER_USERNAME=
DATAFORDELER_PASSWORD=

DATAFORSYNINGEN_TOKEN=
PORT=3000
```

### Start appen

```bash
npm run dev
```

Appen kører på http://localhost:3000
